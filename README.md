# PHP Login System

A secure and functional PHP Login System developed using PHP and MySQL by following a YouTube tutorial as part of internship training tasks.

## 📌 Project Overview

This project includes a complete user authentication system with:

- User Registration
- User Login
- Password Hashing
- Session Management
- Logout Functionality
- Protected Profile Page
- Form Validation
- Modular PHP Structure

The system was developed using core PHP, MySQL, HTML, and CSS.

## ⚙️ How to Set Up and Run the Project

### 1. Install XAMPP

Download and install XAMPP:

https://www.apachefriends.org/


### 2. Clone or Download the Project

Place the project folder inside:

C:\xampp\htdocs\



### 3. Start Apache and MySQL

Open XAMPP Control Panel and start:

- Apache
- MySQL


### 4. Create Database

Open phpMyAdmin:

http://localhost/phpmyadmin


Create a database.

Example:

altitude1-php-login-system


Then import the required SQL table from below drive link
https://drive.google.com/file/d/1v3Lp8uUsSSGkSM4s3YL0ALiiz6JlNbUA/view?usp=sharing

### 5. Configure Database Connection

Open:

includes/dbh.inc.php


Update database credentials if needed.


### 6. Run the Project

Open browser and go to:

http://localhost/altitude1-php-login-system (or your project name)



## 🔐 Features Implemented

- Secure password hashing using `password_hash()`
- Password verification using `password_verify()`
- PHP Sessions for login management
- Logout session destroy
- Protected pages using session validation
